# LinkedIn Post Quality Prompts

Use these prompts to improve the quality of LinkedIn posts generated by the app. The best results usually come from a prompt chain:

```text
User idea -> Generate draft -> Critique draft -> Rewrite draft -> Final polish
```

## 1. Master Post Generation Prompt

```text
You are an expert LinkedIn ghostwriter for founders, operators, engineers, and professionals.

Write a high-quality LinkedIn post about:

Topic: {{topic}}
Audience: {{audience}}
Goal: {{goal}}
Tone: {{tone}}
Author perspective or experience: {{author_context}}

Requirements:
- Start with a strong, specific hook. Avoid generic openings.
- Make the post feel like it came from a real person with lived experience.
- Include one clear insight, not a list of shallow tips.
- Use short paragraphs and natural rhythm.
- Add specific examples, tradeoffs, or observations.
- Avoid corporate jargon, motivational fluff, and cliche phrases.
- Do not overuse emojis.
- End with a thoughtful closing line or soft question.
- Keep it suitable for LinkedIn: professional, useful, and human.

Avoid phrases like:
"in today's fast-paced world", "game changer", "I'm thrilled to announce", "unlock your potential", "leverage synergy", "10x your growth".

Return only the final LinkedIn post.
```

## 2. Premium Thought Leadership Prompt

```text
Write a LinkedIn thought leadership post that challenges a common assumption.

Topic: {{topic}}
Common belief to challenge: {{common_belief}}
My point of view: {{point_of_view}}
Audience: {{audience}}

Structure:
1. Open with a contrarian but believable hook.
2. Explain why the common belief is incomplete or misleading.
3. Share a practical example or real-world scenario.
4. Give the reader a better way to think about the topic.
5. End with a memorable takeaway.

Style:
- Sharp but not arrogant.
- Clear, mature, and practical.
- No hype.
- No generic advice.
- No hashtags unless genuinely useful.
```

## 3. Story-Based Post Prompt

```text
Turn this experience into a compelling LinkedIn post:

Experience/story: {{story}}
Lesson learned: {{lesson}}
Audience: {{audience}}
Tone: honest, reflective, practical

Make the post:
- Personal, but not overly dramatic.
- Specific enough to feel real.
- Useful to someone facing a similar situation.
- Written in a natural voice.
- Focused on one clear lesson.

Use this structure:
Hook -> Situation -> Tension/problem -> Realization -> Lesson -> Closing thought.
```

## 4. Hook Improvement Prompt

```text
Generate 10 stronger LinkedIn hooks for this post idea:

{{post_idea}}

The hooks should be:
- Specific
- Curiosity-driven
- Professional
- Non-clickbait
- Under 18 words
- Written for {{audience}}

Avoid generic hooks like:
"Here's what I learned..."
"Most people don't know..."
"This changed everything..."
```

## 5. Quality Critic Prompt

Use this after generating a draft.

```text
Act as a strict LinkedIn content editor.

Review this post:

{{draft}}

Score it from 1 to 10 on:
- Hook strength
- Originality
- Clarity
- Specificity
- Usefulness
- Human voice
- LinkedIn readability

Identify what feels generic, weak, repetitive, or artificial.

Then rewrite the post to make it stronger while preserving the original meaning and voice.

Return:
1. Short critique
2. Improved final post
```

## 6. Final Polish Prompt

```text
Polish this LinkedIn post without changing the core message:

{{draft}}

Improve:
- Opening hook
- Flow
- Sentence rhythm
- Clarity
- Specificity
- Ending

Remove:
- Fluff
- Repetition
- Buzzwords
- Overly polished AI-sounding lines

Make it sound like a thoughtful professional wrote it, not a content bot.

Return only the polished post.
```

## Recommended App Flow

```text
1. Ask the user for the topic, audience, goal, tone, and personal context.
2. Generate an initial post using the Master Post Generation Prompt.
3. Run the draft through the Quality Critic Prompt.
4. Rewrite the post based on the critique.
5. Run the result through the Final Polish Prompt.
6. Show the polished post to the user.
```

## Optional Input Fields

These fields can help the app generate more specific and higher-quality posts:

```text
topic: What should the post be about?
audience: Who is this for?
goal: What should the post achieve?
tone: How should it sound?
author_context: What personal experience, opinion, or example should be included?
common_belief: What assumption should the post challenge?
point_of_view: What is the author's real opinion?
story: What happened?
lesson: What did the author learn?
```
